# ロボットを動かそう！

##step1
「まえすすめ」のブロックを出してみよう！

「ずっと」のブロックにぴったりはめてみよう！
micro:bit にそうしんしてあるかせてみよう！
```blocks
basic.forever(function () {
    どうさ.まえすすめ()
})

```
##step2
まえにすすめたかな？？
つぎはうしろにあるかせてみよう！

どのブロックをつかいそうかな？？
「まえすすめ」のブロックをよけて、
うしろにすすむためのブロックを「
ずっと」のブロックの中にいれてみよう！

```blocks
basic.forever(function () {
    どうさ.うしろすすめ()
})
```

##step3
うしろに進めたかな？
つぎはカニあるきでよこにすすんでみよう！

すきなカニあるきのブロックを出して，はめこんでみよう！
```blocks
basic.forever(function () {
    どうさ.みぎカニあるき()
})

```
